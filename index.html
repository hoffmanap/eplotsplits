// Project Framework

An Urban Lot Split allows single-family residential parcels to be subdivided into two distinct plots without requiring complex, multi-month public hearings.

By allowing these adjustments "by-right," cities can rapidly unlock missing middle housing—such as duplexes and backyard cottages—increasing infill housing supply within existing neighborhoods while utilizing established municipal utility lines.
